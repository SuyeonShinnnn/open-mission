# 🎊 로또
## 문제
> 3주차 프리코스 문제였던 로또 문제를 Kotlin으로 구현한다. 

- 로또 번호는 수동 또는 자동으로 발급 가능하도록 한다.
- 로또 번호의 숫자 범위는 1~45까지이며, 중복되지 않는 6개의 숫자를 뽑아 로또를 발행한다.
- 당첨 번호 추첨 시 중복되지 않는 숫자 6개와 보너스 번호 1개를 뽑는다.
- 당첨은 1등부터 5등까지 있다. 당첨 기준과 금액은 아래와 같다.
  - 1등: 6개 번호 일치 / 2,000,000,000원
  - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
  - 3등: 5개 번호 일치 / 1,500,000원
  - 4등: 4개 번호 일치 / 50,000원
  - 5등: 3개 번호 일치 / 5,000원
- 로또 1장의 가격은 1,000원이고, 로또 구입 금액을 입력하면 구입 금액에 해당하는 만큼 로또를 발행해야 한다.
- 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 및 수익률을 구한다.
  
## 기능 목록
**📍 로또 구입 금액 입력**
- 사용자로부터 로또를 구입할 금액을 입력받는다.
- 구입 금액은 5,000원 단위로 입력 받고, 5,000원으로 나누어 떨어지지 않으면 예외 처리한다.

**📍 로또 번호 발행**
- 수동/자동 여부를 입력받는다.
- 수동 입력시 로또 번호는 수동으로 입력하여 발행한다
  - 입력은 숫자, 쉼표(,), 공백으로만 이루어져야 한다.
- 한 개의 로또를 발행할 때 6개의 번호를 뽑고, 해당 번호들은 서로 중복되지 않아야 한다.
- 발행된 로또 번호들을 출력한다.

**📍 당첨 번호 & 보너스 번호**
- 로또 당첨 번호와 보너스 번호를 랜덤으로 자동 발행한다.
- 당첨 번호는 1-45까지의 중복되지 않는 숫자를 6개 뽑는다.
- 보너스 번호 또한 1-45까지의 숫자이며, 당첨 번호와 중복될 수 없다.

**📍 당첨 통계**
- 문제에서 정의한 당첨 기준으로 당첨 결과 및 통계를 출력한다.

**🚨 예외 처리**
- 로또 구입 금액 
  - 로또 구입 금액이 5,000보다 작으면 예외 발생
  - 로또 구입 금액이 5,000으로 나누어 떨어지지 않으면 예외 발생

- 로또 번호 생성
  - 로또 번호가 6개가 아닐 때 예외 발생
  - 로또 번호 중 중복된 번호가 있을 때 예외 발생
  - 로또 번호가 1~45까지의 범위에서 벗어나면 예외 발생

- 당첨 번호 & 보너스 번호
  - 당첨 번호로 입력된 값 중에 1~45까지의 숫자, 쉽표(,) 이외의 입력이 있다면 예외 발생
  - 보너스 번호도 1~45 범위 외의 숫자 이외의 입력이 들어오면 예외 발생
  - 보너스 번호가 당첨 번호 중에 중복되는 번호라면 예외 발생

## 실행 예시
**1. 수동 발급**
```
=============================
행운복권 상점에 오신걸 환영합니다🍀
=============================

구입금액을 입력해 주세요.
5000

1. 수동 발급 | 2. 자동 발급
1

** 로또 번호를 수동으로 발급합니다 **
번호를 입력해 주세요. (예: 1, 2, 3, 4, 5, 6)

1번째 로또 번호 입력: 1,2,3,4,5,6

발급한 로또 번호
[1, 2, 3, 4, 5, 6]

당첨번호🍀
[1, 2, 3, 19, 27, 28] + 45

당첨 결과🏆
-----------
5등 당첨
당첨된 로또: [1, 2, 3, 4, 5, 6]
-----------
총 5,000원 수령
수익률: 100%
```

10000
